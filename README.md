# GreenBit: AI-Driven Dark Data Minimization Framework

Detect and eliminate redundant files using semantic analysis, clustering, and carbon tracking.

## Project Structure
- ml_pipeline/ - ML models and semantic analysis
- backend/ - FastAPI server
- frontend/ - React dashboard
- tests/ - Test suites
